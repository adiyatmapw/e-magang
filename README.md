# e-magang

e-Magang is a digital internship recruitment system designed based on the real business flow of internship recruitment processes in Indonesian companies.
This project aims to transform a previously conventional, manual, and non-integrated recruitment process into a centralized and digitalized system that improves efficiency, transparency, and data management.

The system covers key stages of the internship recruitment lifecycle, including candidate registration, data management, and recruitment workflow handling, allowing organizations to manage internship programs more effectively.

### Users
There is one account for HR that manage a whole process, one account for Security Division that manage dicipline security of recruitment. Please use and export the data SQL for using the account.

### Technology Stack
* Backend Framework: Flask
* Database & ORM: SQLAlchemy
* Design & Prototyping: Figma
